# config_tmux

Персональная конфигурация для терминального мультиплексора tmux.

## Описание
Этот репозиторий предназначен для хранения и управления конфигурационными файлами tmux. Настроенная конфигурация позволяет оптимизировать работу в терминале, добавляя удобные горячие клавиши, кастомизацию статусной строки и другие улучшения рабочего процесса.

## Технологический стек
- **Инструмент:** tmux
- **Язык конфигурации:** tmux configuration script (shell-like)

## Установка и запуск

1. Убедитесь, что у вас установлен **tmux** (обычно доступен через менеджер пакетов, например `sudo apt install tmux`).
2. Склонируйте репозиторий:
   ```bash
   git clone git@github.com:qazsedc13/config_tmux.git
   ```
3. Скопируйте файл конфигурации в домашнюю директорию (если файл `.tmux.conf` присутствует в репозитории):
   ```bash
   cp .tmux.conf ~/.tmux.conf
   ```
4. Примените изменения без перезапуска tmux:
   ```bash
   tmux source-file ~/.tmux.conf
   ```

## Примеры использования
- Используйте префикс (по умолчанию `Ctrl+b`) для управления окнами и панелями.
- Конфигурация может включать измененные привязки клавиш для разделения панелей (например, `|` и `-`).

## Структура проекта
- `.tmux.conf` — основной файл конфигурации (ожидается в корне).
- `README.md` — описание проекта.

## Зависимости и требования
- tmux версии 2.1 или выше.
- Unix-подобная операционная система (Linux, macOS, WSL).
